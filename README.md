# Studypool_clone
Studypool Clone: Learning Platform

An interactive, three-tier web application designed to bridge the gap between students and tutors. This platform facilitates academic collaboration by allowing students to post questions and purchase resources while providing tutors with a space to share knowledge and manage academic tasks.

🚀 Overview

The Studypool Clone is a dynamic learning platform that streamlines interaction between students and educators. Students can set budgets and deadlines for their academic queries, while tutors can provide expert solutions and upload study materials like notes and book guides.

✨ Key Features
For Students

Secure Authentication: Dedicated sign-up and sign-in functionality.

Question Posting: Upload academic questions with specific deadlines and budgets for tutor bidding.

Resource Access: Browse and purchase study materials, including detailed notes and book guides.

Tutor Discovery: Connect with qualified tutors across various subjects.

Profile Management: Tools to update personal details and preferences.

For Tutors

Question Management: Browse student-posted questions and select tasks to answer.

Content Uploads: Provide solutions and share study resources (notes/guides) with the community.

Professional Profiles: Manage qualifications and personal details to attract student collaborators.

General Platform Features

Unified Home Page: A user-friendly interface providing quick access for all users.

Secure Transactions: Integrated payment system (Stripe) for safe resource purchases.

Responsive Design: Optimized for seamless experience across devices.

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL

Payment Integration: Stripe

🏗️ System Architecture

The platform follows a three-tier architecture:

Presentation Layer: Responsive UI built with HTML, CSS, and JavaScript.

Application Layer: Backend logic, user authentication, and content management handled by PHP.

Data Layer: MySQL database for secure storage of user data, transactions, and academic resources.
